# Projeto Concurso Agrinho: Equilíbrio entre Produção e Meio Ambiente

## 📝 Descrição do Projeto
Este projeto consiste em um **site educativo** desenvolvido para o concurso Agrinho, com o tema **“Equilíbrio entre produção e meio ambiente”**.  
O objetivo do site é conscientizar sobre a importância de práticas agrícolas sustentáveis e mostrar como é possível produzir alimentos de forma responsável, respeitando o meio ambiente.

O site apresenta informações, exemplos e dicas práticas de sustentabilidade na agricultura, usando uma interface interativa e acessível para todos os públicos.

## 💻 Tecnologias Utilizadas
- **HTML**: Estruturação das páginas e conteúdos do site.  
- **CSS (Style)**: Estilização visual, cores, fontes e layout responsivo.  
- **JavaScript**: Interatividade, animações e recursos dinâmicos.

## ⚙️ Funcionalidades
- Navegação simples e intuitiva entre as seções do site.  
- Informações sobre práticas agrícolas sustentáveis.  
- Exemplos de equilíbrio entre produção e preservação ambiental.  
- Elementos interativos para engajar os visitantes.

## 🚀 Como Executar
1. Baixe ou clone o repositório do projeto.  
2. Abra o arquivo `index.html` no navegador de sua preferência.  
3. Explore as diferentes seções e recursos interativos do site.  

## 🎯 Objetivo do Projeto
Promover a conscientização sobre a sustentabilidade na agricultura e mostrar que é possível **produzir alimentos de forma responsável**, respeitando a natureza e garantindo um futuro mais equilibrado.
